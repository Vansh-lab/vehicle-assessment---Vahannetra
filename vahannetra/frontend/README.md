# VahanNetra Phase 4 Frontend Skeleton

This directory contains the incremental migration target for Phase 4 frontend contracts.

Current Phase 4 artifacts:
- `src/lib/env.ts`: explicit env whitelist reader for browser-safe public env values.
- `src/lib/api.ts`: typed helper methods for phase backend endpoints.
- `src/types/assessment.ts`: shared assessment request/response types.

These files are framework-neutral and intended to be consumed by a future React/Vite or Next.js shell during migration.
